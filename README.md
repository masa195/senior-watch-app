# デジタル同居 - シニア見守りプラットフォーム

離れて暮らす高齢の家族を、デジタルの力で優しく見守るアプリケーションです。

## 🎯 特徴

- **シニアフレンドリーUI**: 大きなボタン、見やすい文字、シンプルな操作
- **リアルタイム見守り**: 「元気です」ボタンで簡単に状況を共有
- **AI気づきエンジン**: 行動パターンの異常を自動検知してアラート
- **家族ダッシュボード**: 離れていても状況を一目で把握

## 🚀 セットアップ

### 必要環境

- Node.js 18以上
- npm または yarn

### インストール

```bash
# 依存関係のインストール
npm install

# 開発サーバー起動
npm run dev
```

### Firebase設定（オプション）

リアルタイム同期機能を使用する場合は、Firebase プロジェクトを作成し、`.env` ファイルを設定してください。

```bash
cp .env.example .env
# .env ファイルにFirebase設定を記入
```

## 📱 機能一覧

### 高齢者側（見守られる方）

| 機能 | 説明 |
|------|------|
| 🟢 元気ですボタン | ワンタップで家族に状況を報告 |
| 🔴 緊急連絡 | 緊急時に即座に家族へ通知 |
| 🍽️ 食事記録 | 食事をしたことを記録 |
| 💊 服薬記録 | 薬を飲んだことを記録 |
| 🌙/☀️ 就寝/起床 | 睡眠状態を記録 |
| 🏠/🚶 在宅/外出 | 外出状態を記録 |

### 家族側（見守る方）

| 機能 | 説明 |
|------|------|
| 📊 ダッシュボード | 高齢者の状態を一目で確認 |
| 📋 活動履歴 | 過去の活動をタイムラインで表示 |
| 🔔 アラート | 異常検知時に通知 |
| 📞 ワンタップ通話 | すぐに電話できる |

### AI気づきエンジン

以下のパターンを自動検知してアラートを送信：

- ⚠️ 長時間チェックインがない
- ⚠️ 食事の間隔が空きすぎている
- ⚠️ 服薬の記録がない
- ⚠️ 外出から長時間戻っていない

## 🛠️ 技術スタック

- **Frontend**: React 18 + TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Backend**: Firebase (Firestore, Authentication)
- **Icons**: Lucide React

## 📁 プロジェクト構造

```
src/
├── components/     # 再利用可能なコンポーネント
├── context/        # React Context (状態管理)
├── hooks/          # カスタムフック
├── lib/            # ユーティリティ、Firebase設定
├── pages/          # ページコンポーネント
└── main.tsx        # エントリーポイント
```
